Domain & Hosting

Register your domain, buy hosting, and install WordPress

1

Register your domain 👤 User Action Required

Your recommended domain is harbourislandinsider.com. Register it at one of these registrars:

Namecheap namecheap.com — approx. $12–14/year. Includes free WhoisGuard privacy. Excellent control panel.
Google Domains domains.google — approx. $12/year. Simple, clean UI, free privacy included. Integrates with Google Search Console.
Tip: Check for first-year coupon codes before checkout — Namecheap often offers domains for $7–9 in the first year.
2

Buy WordPress hosting 👤 User Action Required

Choose managed WordPress hosting — it handles server security, updates, and caching automatically, which directly improves your Google rankings.

WP Engine Starter ~$20/month. Best performance and support. Built-in CDN (Global Edge Security). Ideal if you plan to grow the blog seriously. wpengine.com
SiteGround GrowBig ~$15/month (first year often $3–6 on sale). Excellent speed, free daily backups, free CDN, and great customer support. siteground.com
Recommendation: Start with SiteGround GrowBig — the first-year discount is significant, and it handles everything a travel blog needs. Upgrade to WP Engine later if traffic grows.
3

Point your domain DNS to hosting 👤 User Action Required

After purchasing hosting, your host will give you nameserver addresses (e.g. ns1.siteground.net and ns2.siteground.net). You need to update these at your domain registrar.

At Namecheap:

1. Log in to namecheap.com
2. Click "Domain List" in the left sidebar
3. Click "Manage" next to your domain
4. Under Nameservers, choose "Custom DNS"
5. Enter the nameservers your host provided:
      ns1.yourhostingprovider.net
      ns2.yourhostingprovider.net
6. Click the green tick to save
Important: DNS propagation takes 24–48 hours to complete worldwide. Your site may not resolve immediately — this is normal. You can check progress at whatsmydns.net.

Import Content

Create your 6 articles and 4 static pages in WordPress

1

Understand Posts vs Pages

Posts are your journal articles — they appear in the blog archive, can have categories and tags, and are time-stamped. Pages are static content that doesn't change often (Home, About, Where to Stay).

Your 6 articles = Posts. Your Home, Journal, Where to Stay, and About = Pages.
2

Create the 6 Journal Articles (Posts) ✓ Content Already Written 👤 Upload Required

For each article, go to Posts → Add New and follow this checklist:

  • Paste the article title into the Title field
  • Paste the article body into the content editor (use the Text/Code tab if copying formatted HTML)
  • Set Featured Image (right sidebar): search Unsplash.com for "Harbour Island Bahamas" — download a high-res image and upload it here
  • Categories (right sidebar): assign to journal_category
  • Tags: add 3–5 relevant tags (e.g. "pink sand beach", "Eleuthera", "Bahamas travel")
  • Yoast SEO panel (scroll below editor): enter a unique meta description (150–160 characters summarising the article)
  • Set the URL slug to match the planned URL (e.g. pink-sand-beach-complete-guide)
  • Click Publish

Your 6 articles and their URL slugs:

Article 1 Ultimate Cruiser's Guide to Valentine's Marina — ultimate-cruisers-guide-valentines-marina
Article 2 Pink Sand Beach Complete Guide — pink-sand-beach-complete-guide
Article 3 Top 7 Dive Sites at Harbour Island — top-7-dive-sites-harbour-island
Article 4 Best Restaurants Harbour Island 2026 — best-restaurants-harbour-island-2026
Article 5 Getting to Harbour Island — getting-to-harbour-island
Article 6 25 Things to Do in Harbour Island — 25-things-to-do-harbour-island

SEO Setup

Submit your site to search engines and configure analytics

1

Upload sitemap.xml and robots.txt ✓ Files Already Built 👤 Upload Required

Both files are in the /seo/ folder of this project. They need to live in the root of your website so they're accessible at:

https://harbourislandinsider.com/sitemap.xml
https://harbourislandinsider.com/robots.txt

Upload them via your hosting file manager or FTP to the public_html/ directory (or www/ — whichever is your web root).

SiteGround: cPanel → File Manager → public_html → Upload
WP Engine: SFTP to /sites/your-site/public/
2

Submit to Google Search Console 👤 User Action Required 

1. Go to search.google.com/search-console
2. Click "Add Property" → enter: https://harbourislandinsider.com
3. Choose "URL prefix" verification method
4. Select "HTML tag" option
5. Copy the meta tag Google provides, e.g.:
      <meta name="google-site-verification" content="XXXXXXXXXXXX" />
6. In WordPress: Yoast SEO → Settings → Webmaster Tools
7. Paste the content value (just the alphanumeric code) into "Google"
8. Save, then click "Verify" in Search Console
9. Once verified: Sitemaps → enter "sitemap.xml" → Submit
3

Submit to Bing Webmaster Tools 👤 User Action Required 

1. Go to bing.com/webmasters
2. Sign in with a Microsoft account
3. Add your site: https://harbourislandinsider.com
4. Verify via the XML file method or auto-import from Google Search Console
5. Sitemaps → Submit → enter: https://harbourislandinsider.com/sitemap.xml
4

Set Up Google Analytics 4 👤 User Action Required 

1. Go to analytics.google.com
2. Click Admin (gear icon, bottom left)
3. Create Account → Account Name: "Harbour Island Insider"
4. Create Property → Property Name: "harbourislandinsider.com"
5. Select: Web → enter your URL → create Web Stream
6. Copy your Measurement ID (format: G-XXXXXXXXXX)
7. In WordPress: Settings → Harbour Island Insider → GA4 ID
8. Paste your Measurement ID → Save
5

Verify GA4 Is Tracking

After saving your GA4 ID, open a new browser tab and visit your website. Then go back to Google Analytics and check:

Path Reports → Realtime → you should see "1 user in last 30 minutes"
If you don't see yourself in Realtime, clear your browser cache and try again. Ad blockers can block GA4 — try an incognito window without extensions.

Name:        Island Dispatch

3

Get Your API Key 👤 User Action Required 

Mailchimp → Click your profile icon (top right)
→ Account & Billing
→ Extras → API Keys
→ Create A Key
→ Copy the key (it looks like: abc123def456...<datacenter>)
Keep this key private. Never share it or commit it to a public code repository. Store it in a password manager.
4

Get Your Audience ID (List ID) 👤 User Action Required 

Audience → Manage Audience → Settings
→ Audience name and defaults
→ Scroll to "Audience ID" — copy this value
(It looks like: a1b2c3d4e5)
5

Enter Credentials in WordPress 👤 User Action Required 

WordPress Admin → Settings → Harbour Island Insider
→ Mailchimp API Key: [paste your key]
→ Mailchimp List ID: [paste your audience ID]
→ Save Settings

Your newsletter sign-up form will now capture subscribers directly into your "Island Dispatch" Mailchimp audience.

→ Enable double opt-in: Yes
